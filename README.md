# Superstore Sales Analysis

## Project Overview
This project explores retail sales data to identify trends in sales, profitability, customer behavior, and regioanal performance using python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Sales increased steadily from 2015 to 2017.
- November records the highest monthly sales.
- The West region generates the highest revenue.
- The Consumer segment generates the highest profit.
- Higher discounts reduce profitability.
- Sean Miller generated the highest sales.
- Tamara Chand and Raymond Buch generated the highest profit.
